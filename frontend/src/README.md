# Blue Lotus Runtime (frontend/src)

This folder contains the core runtime for the Blue Lotus app builder:

- `App.jsx` — root application component
- `main.jsx` — Vite entry point
- `index.css` — global styles
- `components/` — runtime UI components
- `pages/` — page rendering system
- `layout/` — app layout wrapper
- `router/` — simple runtime router
- `state/` — project context provider
- `utils/` — helper utilities

This runtime is clean, deterministic, and free of legacy code.
