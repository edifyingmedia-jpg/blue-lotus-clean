# Blue Lotus Builder

This folder contains the Builder interface for creating and editing apps inside Blue Lotus.

## Files

- `Builder.jsx` — Entry point for Builder mode
- `BuilderApp.jsx` — Main Builder UI container
- `ComponentPanel.jsx` — Left-side panel for adding components
- `PageRenderer.jsx` — Live preview of the current page

The Builder is isolated from the runtime and uses the same global project state.
